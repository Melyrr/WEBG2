Testez votre connaissances en Java !

  1. Qu'affiche System.out.println("1"+"2") ?
    • Réponse : 12
    • Explications : Le + entre 2 chaines les concatènes.
  2. Qu'affiche System.out.println(1+2) ?
    • Réponse : 3
    • Explications : Le + entre 2 entier les addtionne.
  3. Qu'affiche System.out.println("1+2=" +1+2) ?
    • Réponse : 1+2=12
    • Explications : Le premier + concatène ( en transformant le 1 en chaine).
      Idem pour le second +
  4. Qu'affiche System.out.println("1+2=" +(1+2)) ?
    • Réponse : 1+2=3
    • Explications : Les parenthèes impliquent qu'on fasse le second + (addition).
      Ensuite, le premier + concatène.
  5. Si ok est un booléan initialisé à false, que va faire if (ok=true) System.out.println("OK!"); ?
    • Réponse : Afficher OK!
    • Explications : La condition est enfaite une assignation qui met ok à vrai.
      Du coup, la condition est vraie.
